# 자바 게임 구현 프로젝트

JavaProject Game


[작품 이름] 

토끼와 거북이의 모험 (RPG) 

[작품 소개]

주인공인 토끼 와 거북이 중 주인공 케릭터를 고르고 퀘스트를 통해 게임을 진행합니다.

사냥터에서 레벨업을 할수있으며 공복도(피로도)를 관리하며 가지고있는 돈으로 장비 및 음식을 먹거나 구입 가능합니다.

퀘스트를 진행하며 시련을 이겨내고 궁국적으로 마지막 퀘스트를 클리어시 게임클리어가 되는 성장형 RPG 입니다.
